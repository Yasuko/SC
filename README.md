提供機能

- スクリーンショットの撮影
----


----
## Build（開発）
----
開発環境
- Angular：8.0以上
- NodeJs ：10.0以上
- Httpサーバー（静的コンテンツ参照用）
<br>


ダウンロード後に、プロジェクトフォルダ内で

`npm install`

関連パッケージがインストールされる<br>
インストール完了後

`ng build`

ブラウザより下記にアクセス<br>
静的コンテンツ  ：https://localhost/プロジェクト/dist/<br>
